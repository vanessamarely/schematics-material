Schematics in Angular Material

Generate pre-made layout and template snippets of Material Components. Schematics help to make creating Material applications easier.

Install Schematics

It come packaged with Angular Material. If you run the following code line, it will automatically install Angular Material for you and run the install schematic.

ng add @angular/material

Generator: Navigation

It will create a new component that includes a toolbar with the app name and the side nav responsive based on Material breakpoints.

ng generate @angular/material:material-nav --name 'component-name'

Generator: Dashboard

It will create a new component that contains a dynamic grid list of cards.

ng generate @angular/material:material-dashboard --name 'component-name'<

Generator: Table

It will create a new table component pre-configured with a datasource for sorting and pagination.

ng generate @angular/material:material-table --name 'component-name'
